## What is FrankerFaceZ?

FrankerFaceZ (FFZ) is a browser extension and userscript that enhances the Twitch experience. It ships with custom emote support, a flexible settings system, and a full-featured **add-on framework** that allows third-party developers to extend and customize Twitch in virtually unlimited ways.

FFZ has been in active development since 2013 and powers features used by millions of Twitch viewers and streamers every day.

## How add-ons work

FrankerFaceZ exposes a stable public API through `window.ffz` and `window.FrankerFaceZ`. Add-ons register themselves into the module tree and gain access to all FFZ systems through dependency injection.

```typescript addon.ts
import { Addon } from 'utilities/addon';

class MyAddon extends Addon {
  static info = {
    id: 'my-addon',
    name: 'My Add-on',
    author: 'YourName',
    description: 'Does something cool on Twitch.'
  };

  async onEnable() {
    const chat = this.resolve('chat');
    this.log.info('My add-on is enabled!');
  }
}

MyAddon.register();
```
