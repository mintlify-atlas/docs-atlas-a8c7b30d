`FrankerFaceZ` is the top-level class that bootstraps the entire extension. It extends [`Module`](/api/module) and is instantiated once when the extension loads. The live instance is exposed as `window.ffz` and the class itself as `window.FrankerFaceZ`.

```typescript
import Module, { State } from 'utilities/module';

class FrankerFaceZ extends Module { ... }

window.FrankerFaceZ = FrankerFaceZ;
window.ffz = new FrankerFaceZ();
```

## Static Methods

### `get()`

```typescript
static get(): FrankerFaceZ
```

Returns the current singleton instance of `FrankerFaceZ`. Equivalent to reading `FrankerFaceZ.instance`.

```typescript
const ffz = FrankerFaceZ.get();
console.log(ffz.version_info.toString()); // e.g. "4.21.0"
```

## Static `utilities` Namespace

`FrankerFaceZ.utilities` is a plain object that re-exports every utility module. Add-ons should use these exports instead of importing from the module paths directly, so they stay compatible across FFZ versions.

```typescript
static utilities = {
  addon:      Utility_Addons,
  blobs:      Utility_Blobs,
  color:      Utility_Color,
  constants:  Utility_Constants,
  dialog:     Utility_Dialog,
  dom:        Utility_DOM,
  events:     Utility_Events,
  fontAwesome: Utility_FontAwesome,
  graphql:    Utility_GraphQL,
  logging:    Utility_Logging,
  module:     Utility_Module,
  object:     Utility_Object,
  time:       Utility_Time,
  tooltip:    Utility_Tooltip,
  i18n:       Utility_I18n,
  filtering:  Utility_Filtering,
};
```

## Instance Methods

### `generateLog()`

```typescript
async generateLog(): Promise<string>
```

Collects diagnostic output from every registered module and returns it as a formatted plain-text string. Each module's section is separated by a dashed rule. The initialization log (captured during startup) is prepended as the first section.

Module `generateLog` calls are individually wrapped in a 5-second timeout. If a module times out or throws, its section contains the error message instead of diagnostic data.

```typescript
const log = await FrankerFaceZ.get().generateLog();
console.log(log);
// initialization
// -------------------------------------------------------------------------------
// [0:00:01] info | core: Initialization complete in 312.00000ms.
//
// settings
// -------------------------------------------------------------------------------
// { ... }
```

<ResponseField name="return" type="Promise<string>">
  A formatted multi-section diagnostic string.
</ResponseField>
