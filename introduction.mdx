## What is FrankerFaceZ?

FrankerFaceZ (FFZ) is a browser extension and userscript that enhances Twitch with custom emotes, a flexible settings system, and a full-featured add-on framework. Third-party developers can build add-ons that hook directly into the same module tree that FFZ itself uses тАФ with access to settings, chat rendering, badges, emotes, and more.

FFZ is open source under the Apache 2.0 license. The current version is **4.81.0**.

## Architecture overview

FFZ is structured as a tree of **modules**. Every subsystem тАФ chat, emotes, settings, i18n, the add-on manager тАФ is a `Module` instance mounted at a named path in a shared registry. Add-ons are mounted under the `addon.<id>` path namespace and participate in the same lifecycle.

```
FrankerFaceZ (root)
тФЬтФАтФА settings          SettingsManager
тФЬтФАтФА experiments       ExperimentManager
тФЬтФАтФА i18n              TranslationManager
тФЬтФАтФА socket            SocketClient
тФЬтФАтФА pubsub            PubSubClient
тФЬтФАтФА site              Site
тФВ   тФФтФАтФА ...           (chat, player, player-bar, ...)
тФФтФАтФА addons            AddonManager
    тФЬтФАтФА addon.my-addon  MyAddon       тЖР your add-on
    тФФтФАтФА addon.other     OtherAddon
```

Every node in the tree is an instance of [`Module`](/api/module). Modules declare their dependencies with `inject()` or `requires`, and FFZ ensures they are enabled in the correct order before calling `onEnable()`.

## Public API surface

Once FFZ is loaded, two globals are available on every Twitch page:

| Global | Type | Description |
|---|---|---|
| `window.ffz` | `FrankerFaceZ` | The running FFZ instance. |
| `window.FrankerFaceZ` | `typeof FrankerFaceZ` | The class itself, exposing `FrankerFaceZ.get()` and `FrankerFaceZ.utilities`. |

Add-ons access the instance indirectly through `Addon.register()`, which calls `window.FrankerFaceZ.get()` internally:

```typescript
// src/utilities/addon.ts
static register(id?: string | AddonInfo, info?: AddonInfo, version?: string) {
  // ...
  const ffz = window.FrankerFaceZ.get();
  ffz.register(`addon.${id}`, this);
}
```

`FrankerFaceZ.utilities` is a namespace object exposing every utility module for add-on authors:

```typescript
// src/main.ts
static utilities = {
  addon:      Utility_Addons,
  blobs:      Utility_Blobs,
  color:      Utility_Color,
  constants:  Utility_Constants,
  dialog:     Utility_Dialog,
  dom:        Utility_DOM,
  events:     Utility_Events,
  fontAwesome: Utility_FontAwesome,
  graphql:    Utility_GraphQL,
  logging:    Utility_Logging,
  module:     Utility_Module,
  object:     Utility_Object,
  time:       Utility_Time,
  tooltip:    Utility_Tooltip,
  i18n:       Utility_I18n,
  filtering:  Utility_Filtering
};
```
